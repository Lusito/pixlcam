# restrictToBounds

▸ restrictToBounds(input, min, max, viewportSize): number

Restrict a camera position to be contained within the specified bounds according to its viewport size. If the width of the bounds is smaller than the viewportSize, the center will be returned.

# Parameters

Name Type Description
input number The value to restrict.
min number The minimum value of the bound.
max number The maximum value of the bound.
viewportSize number The size of the viewport. Min and max will be reduced by half of this value.

# Returns

number

The restricted value.

# snapToPixel

▸ snapToPixel(value, viewportSize, zoom): number

Snap a value to a pixel depending on viewport size and zoom level. If viewport size is divisible by 2, the zoomed value will be snapped to integers. Otherwise the zoomed value will be snapped to .5 values instead.

# Parameters

Name Type Description
value number The value to snap.
viewportSize number The viewport size.
zoom number The zoom level of the camera.

# Returns

number

The adjusted value.
