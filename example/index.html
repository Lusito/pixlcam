Camera2D library: demo

This is a simple demonstration of the Camera2D library. Check out the example source-code.

Use arrow keys to move and use WASD to aim (when using influenced camera).

Press T to teleport to a random location and Space to shoot a rocket (when using influenced camera and aiming).
