PixlCam: demo

This is a demonstration of the PixlCam library for 2D cameras. Check out the example source-code. The influenced camera is inspired by the camera from Insanely Twisted Shadow Planet

Use arrow keys to move and use WASD to aim (when using influenced camera).

Press T to teleport to a random location and Space to shoot a rocket (when using influenced camera and aiming).
